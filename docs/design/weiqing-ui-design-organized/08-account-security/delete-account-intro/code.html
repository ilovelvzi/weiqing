账号注销

注销账号前，请先了解这些

注销账号会进入 30 天冷静期。冷静期内你可以恢复账号，也可以导出自己的历史数据。

冷静期结束后，账号数据将进入删除流程，届时无法再恢复或导出。

将被删除的数据

monitor_weight 体重记录
favorite 健康数据
flag 目标数据
restaurant 饮食线索
analytics 周报/月报
smart_toy AI 鼓励缓存
image上传图片
rss_feed动态
chat_bubble评论
thumb_up点赞

如果你发布过动态，相关文本、图片、评论和点赞也会被删除。已删除记录不会进入导出文件，也不会被恢复。

不会被微轻删除的数据

health_and_safety Apple Health 原本已有的数据
policy法律或安全审计要求保留的去标识化日志
