## zod TS2307 排障结论

现象：
@weiqing/shared 中 import { z } from 'zod' 报 TS2307。

最终原因：
zod@3.25.0 的 package exports 指向 dist/esm/index.d.ts / dist/esm/index.js，
但当前安装包中目标文件不存在，导致 TypeScript bundler resolution 解析失败。

排除项：
- zod 依赖已安装
- @weiqing/shared package.json 已声明 zod
- tsconfig.base.json 的 moduleResolution: "Bundler" 已生效
- packages/shared/tsconfig.json 不是主因

修复：
升级 zod 到稳定 patch 版本，例如：

pnpm --filter @weiqing/shared add zod@3.25.76
